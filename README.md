Fitness Activity Mobile App (Figma UI)

A simple and modern fitness tracking mobile app UI designed in Figma. The app helps users monitor their daily activity through clean, minimal screens.

Features
- Track steps, **calories**, and **heart rate**
- Male & female dashboard versions
- Monthly activity history
- Profile and progress screens
- Clean, minimal, data-focused layout
